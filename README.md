# ISIT-Инвентарь - демонстрационная версия системы

Демонстрационная версия системы учёта оборудования кафедры ИСиТ Заполярного Государственного Университета. Все сведения в системе вымышленные и предназначены только для ознакомления с интерфейсом.

## Вход в систему

На странице входа укажите логин/электронную почту и пароль из таблицы.


| Роль                      | Логин                 | Пароль      |
| ------------------------- | --------------------- | ----------- |
| Администратор             | `admin@email.com`     | `admin`     |
| Материально ответственный | `inventory@email.com` | `inventory` |
| Лаборант                  | `laborant@email.com`  | `laborant`  |
| Пользователь              | `user@email.com`      | `user`      |


У разных ролей разный набор возможностей (заявки, списание, панели редактирования и др.). Чтобы увидеть систему целиком, войдите под несколькими учётными записями.
